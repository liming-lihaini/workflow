本模块以 `com.flow.engine.FlowEngineApplication` 为启动入口，按分层 + 领域子包组织：
- 表现层：`controller/` 暴露业务 API（流程、实例、任务），`controllers/` 提供管理面能力（用户、角色、权限、字典、Webhook）。
- 领域核心：`engine/FlowEngine` 驱动状态机推进流程；`node/` 通过 `NodeHandlerRegistry` + `AbstractNodeHandler` 抽象注册式节点处理器，内置 `Start/End/UserTask/ServiceTask/ScriptTask/Gateway/SubProcess` 等实现，由 `NodeHandlerAutoConfiguration` 自动发现。
- 持久化：`entity/` + `mapper/` 使用 MyBatis-Plus 映射到 SQLite（`schema.sql` 初始化）。
- 横切：`common/` 提供全局异常、Result 包装、RequestIdFilter、OpLog 注解及 AOP 日志；`config/` 集中装配缓存、MyBatis-Plus、WebMvc、Webhook 定时调度等。
- 事件：`event/` 定义流程/节点生命周期事件，`listener/` 中的 `WebhookEventListener` 将其桥接到 `webhook_event` 模块的异步 HTTP 回调。
- 集成点：`task_center` 模块通过调用 `ProcessInstanceService` / `TaskService` 驱动引擎；`data_model` 模块提供 JSON 模型解析供 `ProcessJsonParser` 消费；`webhook_event` 模块监听同一 Spring 事件总线完成外部通知。
