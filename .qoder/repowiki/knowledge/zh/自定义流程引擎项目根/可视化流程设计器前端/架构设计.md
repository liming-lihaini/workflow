单页 React 应用（Vite + React 18），入口 `src/main.jsx` 挂载 `App.jsx`。
- 状态与编排集中在 `App.jsx` 中的 `Designer` 组件：使用 `useNodesState`/`useEdgesState` 管理画布数据，通过 `ReactFlowProvider` 提供上下文；所有交互（拖放新增、连线、删除、校验、保存）在此集中处理。
- 视图层拆为三个无状态子组件：`components/FlowNode.jsx`（统一渲染各类节点，根据 `nodeType` 动态选择颜色/图标）、`components/NodePalette.jsx`（左侧节点面板，按分组渲染可拖拽项）、`components/ConfigPanel.jsx`（右侧属性面板，按选中对象类型渲染表单）。
- 领域模型与 UI 行为解耦在 `src/nodeTypes.js`：以声明式 schema 描述每种节点类型的 label/color/icon/group/schema，同时驱动 NodePalette 展示、FlowNode 渲染以及 ConfigPanel 表单生成，形成「schema → 面板/节点」的双向映射。
- 网络层封装于 `src/api.js`：基于 axios 创建带 baseURL `/api/v1`、请求拦截注入 Bearer token、响应拦截统一处理 `code` 字段与 401 清理 token 的实例，并暴露 `process/definitions` 资源 REST 方法；`App.jsx` 仅调用这些函数，不直接依赖 axios。
- 依赖方向：`App.jsx` → 子组件 / nodeTypes / api；子组件只读依赖 nodeTypes，无横向依赖；对外边界仅为 `/api/v1/process/definitions*`。
