单页应用，入口 `src/main.js` 通过 `createApp` 挂载 Pinia、vue-router 与全局 Antd 组件库，并注入全局样式变量。
- 路由层：`src/router/index.js` 集中声明所有路由，使用 `component: () => import(...)` 实现按页面懒加载；根路由以 `BasicLayout.vue` 为壳，子路由统一挂入 `/dashboard`、`process/*`、`task/*`、`system/*`、`monitor` 等目录结构，并通过 `meta.title` 驱动侧边菜单标题；`beforeEach` 守卫基于 `localStorage.token` 做登录态校验与 401 自动跳转。
- API 层：`src/api/` 下每个业务域一个文件（admin/auth/dict/log/monitor/process/system/task），均 `import request from './request'` 调用，由 `request.js` 统一封装 axios 实例（baseURL=/api/v1、超时 15s），并在请求拦截器自动注入 `Authorization: Bearer <token>`，响应拦截器对 `{code,data,message}` 统一成功/失败处理及 401/403 提示。
- 状态层：`src/stores/user.js` 使用 Pinia 持久化用户信息（配合 token）。`components/` 目录为空，当前页面逻辑直接内聚在 `views/` 对应 `.vue` 文件中，尚未抽取通用业务组件。
- 依赖方向：views → router/meta → api/* → api/request → axios；layout 仅负责布局壳，不持有业务状态。
